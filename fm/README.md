# Web文件管理器 


## 文件列表
1. `index.php`          - 后端处理
2. `styles.css`         - 样式定义
3. `editor.js`          - 文件编辑器
4. `main.js`            - 核心文件管理
5. `kebab-case.js`      - 前端交互功能
6. `config.php`         - 用户配置文件模板
7. `pas_hash.php`       - 简化版密码哈希生成器
8. `password_hash.php`  - 密码哈希生成器
9. `index单文件.php`     - 单文件版本 只需单文件就可运行

## 项目概述
这是一个Web文件管理器，提供类似桌面文件管理器的体验。项目采用前端JavaScript + 后端PHP的架构，通过AJAX实现无刷新操作。

## 主要功能模块

## 🔐 认证与安全

| 功能 | 说明 |
|------|------|
| 登录认证 | 用户名 + bcrypt 密码验证 |
| 登录锁定 | 5次失败后锁定15分钟，防暴力破解 |
| CSRF 保护 | 所有操作带 Token 验证 |
| 路径安全检查 | `fm_is_safe_path()` 防目录穿越 |
| 上传扩展名过滤 | 禁止上传 php/exe/bat 等危险文件 |
| ZIP 解压安全检查 | 防止 Zip Slip 路径穿越攻击 |
| 外部配置支持 | 可通过 `config.php` 覆盖默认配置 |

## 🏗️ 技术特点

- **单文件部署** - 只需一个 `index.php`，无依赖
- **纯后端渲染** - PHP 输出 HTML，无需前端框架
- **AJAX 操作** - 所有文件操作通过 AJAX 异步完成
- **配置分离** - 支持 `config.php` 外部配置，更新不丢失
- **跨平台** - 支持 Windows / Linux

## 实现方法分析

### 前端技术栈
- **JavaScript（ES6+）**：核心交互逻辑
- **AJAX（Fetch API）**：与后端通信
- **DOM操作**：动态更新界面
- **本地存储**：保存用户偏好和状态
- **ACE编辑器**：代码编辑功能
- **拖放API**：文件上传和移动
- **History API**：SPA路由

### 后端技术栈
- **PHP（原生）**：后端逻辑处理
- **文件操作函数**：处理文件上传、移动等
- **压缩/解压库**：ZipArchive、PharData
- **系统命令调用**：处理7Z、RAR格式
- **会话管理**：用户认证和状态保持
- **JSON响应**：AJAX请求返回数据

### 安全技术
- **路径遍历防护**：`fm_is_safe_path()`函数校验
- **CSRF防护**：Token验证
- **文件类型限制**：禁止上传可执行文件
- **权限控制**：用户认证后才能操作
- **输入过滤**：清理用户输入

## 技术亮点

1. **SPA架构**：无刷新操作，用户体验好
2. **完整功能**：涵盖文件管理各方面需求
3. **安全性**：多层安全防护
4. **响应式设计**：适配桌面和移动端
5. **可扩展性**：模块化设计，易于添加新功能
6. **用户体验**：键盘快捷键、拖拽操作、进度提示等
